# 나의 할일 관리 앱

## 1. 앱소개
사용자가 하루 동안 해야 할 과업을 체계적으로 기록하고 관리할 수 있도록 돕는 **'웹 기반 할 일 관리(To-Do List) 프로그램'**입니다.

## 2. 사용한 AI
제미나이, 챗GPT

## 3. 처음 입력한 프롬프트
HTML, CSS, JavaScrit을 하나의 index.html 파일에 넣어서 할 일 관리앱을 만들어줘 할일을 입력하고 추가할 수 있고, 완료 표시와 삭제 기능이 있었으면 좋겠어.초보자도 이해할 수 있게 코드를 너무 복잡하게 만들지 말아줘.
