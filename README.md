# 나의 할일 관리 앱

## 1. 앱소개
사용자가 하루 동안 해야 할 과업을 체계적으로 기록하고 관리할 수 있도록 돕는 **'웹 기반 할 일 관리(To-Do List) 프로그램'**입니다.

## 2. 사용한 AI
제미나이, 챗GPT

## 3. 처음 입력한 프롬프트
HTML, CSS, JavaScrit을 하나의 index.html 파일에 넣어서 할 일 관리앱을 만들어줘 할일을 입력하고 추가할 수 있고, 완료 표시와 삭제 기능이 있었으면 좋겠어.초보자도 이해할 수 있게 코드를 너무 복잡하게 만들지 말아줘.

## 4. 수정 요청한 내용
음 좋았어. 이제 디자인을개선할건데 index.html 파일에 배경색을 하늘색, 화면은 가운데 카드모양으로,보기 좋게 사이드에는 여백을 만들어줘. 입력창과 버튼을 보기 좋게 배치하고, 글자와 색상도 조화롭게, 사용자가무엇을 해야하는지 알 수 있는 안내문구도 조화롭게 꾸며줘

## 5. 어려웠던 점 또는 배운 점
어려웠던 점 
- 처음으로 직접 웹페이지를 만들다 보니, 머릿속으로 구상한 기능을 구현하기 위해 코딩을 어디서부터 어떻게 시작해야 할지 막막했다. 화면 레이아웃을 구성하는 과정에서 원하는 대로 디자인이 나오지 않아 어려움을 겪었고, 개발 도중 오류가 발생했을 때 내가 무엇을 모르는지 정확히 몰라 검색창이나 AI에게 어떻게 질문 해야하는지 막막했던 어려움이 있었다.

배운 점
- 문제를 해결하기 위해 복잡한 코드를 기능별로 잘게 쪼개어 생각하는 법을 연습했다. 내가 직면한 오류의 증상을 명확히 정리해 질문하는 과정을 통해, AI에게는 정확하게 질문하는 능력(프롬프트 기획 및 검색 능력)이 코딩 실력만큼이나 중요하다는 것을 깨달았다. 막막했던 막대 코딩에서 벗어나, 모르는 것을 주도적으로 찾아내고 해결하는 핵심적인 문제해결력을 기를 수 있었다.
